# 🎧 RustTune – High Performance Music Streaming App

RustTune is an open-source music streaming application built using **Rust**, focused on performance, safety, and scalability. This project demonstrates how Rust can be used to build fast and efficient web applications.

---

## 🚀 Features

- 🎵 Music playback system
- 📂 Playlist creation and management
- 🔍 Search functionality
- ⚡ High-performance backend using Rust
- 📱 Responsive and user-friendly UI
- 🔐 Secure and efficient architecture

---

## 🛠️ Tech Stack

- **Backend:** Rust (Actix Web)
- **Frontend:** HTML, CSS, JavaScript
- **Version Control:** Git & GitHub

---

## 📌 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/rust-tune.git
cd rust-tune
```

---

### 2. Install Rust

Download and install Rust from:

https://rustup.rs/

Verify installation:

```bash
rustc --version
cargo --version
```

---

### 3. Run the Project

Inside the project directory, run:

```bash
cargo run
```

If the build is successful, the server will start at:

```txt
http://127.0.0.1:8000
```

Open the above URL in your browser to view the application.
